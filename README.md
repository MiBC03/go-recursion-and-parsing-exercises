# Go Recursion and Parsing Exercises

Este repositório contém implementações em Go de dois exercícios acadêmicos:

1. Desenho de árvores binárias com cálculo de posicionamento dos nós.
2. Implementação de um parser descendente recursivo para uma linguagem simples.

## Objetivo
Demonstrar a tradução correta de conceitos tradicionalmente abordados em linguagens
funcionais, como recursão e árvores de sintaxe abstrata, para a linguagem Go.

## Tecnologias
- Go

## Conceitos Abordados
- Recursão
- Estruturas de dados
- Árvores binárias
- Parsing descendente recursivo
